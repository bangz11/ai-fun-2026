# Indoor Plant, Flower & Vegetable Recommendation AI 🌱

This project recommends plants that can grow well in a dormitory or indoor environment.

## Features
- Recommend indoor plants, flowers, and vegetables
- Uses sunlight and watering preference
- Simple AI decision system

## Dataset
The dataset includes:
- Indoor plants
- Flowers
- Vegetables that can be grown in small spaces

## Technologies
- Python
- Pandas
- Jupyter Notebook

## How it works
The user enters:
- Sunlight level (low / medium / high)
- Watering ability (low / medium / high)

The system then recommends suitable plants.
